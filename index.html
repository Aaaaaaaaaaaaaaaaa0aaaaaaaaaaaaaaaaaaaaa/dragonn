Overview of DragoNN

The dragonn package implements deep neural networks (DNNs) for predictive modeling of regulatory genomics, feature importance methods such as DeepLIFT and in-silico mutagenesis to interpret DNNs, and sequence simulations systematically assess model development and interpretation.

Installation

You can install dragonn from anaconda with:

conda install -c kundajelab dragonn
